# kypanz-tts-offline

This script is an offline way to listen messages based in the input

## Requirements

install mplayer

```
sudo apt install mplayer
```

others

```
node v20.9.0
npm 10.1.0
```

## Configuration

create a `.env` file and add the next, also you can change the values :

```
SPEED='2'
LANGUAGE='es-us'
```
